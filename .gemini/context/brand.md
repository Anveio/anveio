# Brand Context for Cline

**Brand Name:** Anveio

**Description:** Anveio is a B2B platform built with modern web technologies including Next.js, Convex, Clerk, and Stripe. It provides full-stack serverless web application capabilities designed for rapid development and scalability. The platform leverages best-in-class managed services for database, authentication, and payments.

**Key Technologies:**
- Frontend: Next.js (React)
- Backend: Convex
- Authentication: Clerk
- Payments: Stripe
- UI: shadcn/ui, Tailwind CSS

**Project Context:** This is the foundational codebase for Anveio, a B2B SaaS solution. All references to the original "SaaS Application" should now be updated to reflect the brand "Anveio".

**Logo:** No changes required to the logo at this time, as per the user's instruction.

**Pricing Tiers:**

*   **Basic:** $29/month. Includes 5 products, up to 1,000 subscribers, and basic analytics.
*   **Pro:** $79/month. The most popular plan, it includes 25 products, up to 10,000 subscribers, advanced analytics, and 24-hour support response time.
*   **Enterprise:** $149/month. Includes unlimited products and subscribers, advanced analytics, 1-hour dedicated support response time, and marketing automations.

This context file should be used in future executions of Cline to ensure branding consistency and understanding of the application's purpose and technology stack.
